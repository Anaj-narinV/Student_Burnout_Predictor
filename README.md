# Student Burnout Predictor

## Overview
This is a machine learning-based web application that predicts student burnout levels based on academic pressure, study habits, and lifestyle factors. The objective of this project is to identify early signs of burnout and provide data-driven insights for awareness and prevention.

## Features
- Predicts student burnout level in real-time  
- Machine learning model for prediction  
- Interactive dashboard with data visualization  
- Simple and user-friendly web interface  
- Input-based prediction using academic and lifestyle parameters  

## Tech Stack
- Python  
- Streamlit  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib / Seaborn
